    1. Can you post the commands you used to create by_package?

    2. Go into play() mode in swirl and try testing the summarize command with one argument at a time, then start adding arguments back in.  E.g. start by testing:
    summarize(by_package, count = n())
    summarize(by_package, unique = n_distinct(ip_id))
    summarize(by_package, countries = n_distinct(country))
    summarize(by_package, avg_bytes = mean(size))
    This may pinpoint the error further.

    You might also try running the summarize statements bare to the console, as shown above, vs. assigning it to pack_sum and then printing pack_sum -- this is another way of trying to tear down the statement to its smallest components that don't give an error, and then build it back up piece by piece until you find where the error happens.
